# Acode Dev Master Installer

**Acode Dev Master Installer** je modulární nástroj pro instalaci a správu mobilního i desktopového vývojového prostředí. Poskytuje jednotné workflow pro analýzu kódu, AI review, správu pluginů a integraci s různými platformami (Termux, WSL, Raspberry Pi, macOS).

## Rychlý start
1. Ulož obsah do adresáře `$HOME/.acode_dev_master` nebo do `Acode_Dev_Master/`.
2. Nastav spustitelnost:
   ```bash
   chmod +x master_install.sh
   ./master_install.sh
